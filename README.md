# bayan — утилита поиска дублей файлов

## Описание

Утилита сканирует указанные директории и находит файлы с одинаковым содержимым.
Сравнение ведётся поблочно с ленивым чтением: каждый блок читается с диска
не более одного раза и только тогда, когда есть реальный кандидат на сравнение.

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Использование

```
bayan [параметры]
```

## Параметры

| Параметр           | Краткая форма | Описание                                              | По умолчанию |
|--------------------|---------------|-------------------------------------------------------|--------------|
| `--dirs`           | `-d`          | Директории для сканирования (можно несколько)         | —            |
| `--exclude`        | `-e`          | Директории для исключения (можно несколько)           | —            |
| `--level`          | `-l`          | Глубина сканирования (-1 = рекурсивно, 0 = только директория) | -1  |
| `--min-size`       | `-m`          | Минимальный размер файла в байтах                     | 1            |
| `--mask`           | `-n`          | Маски имён файлов (например `*.txt`), без учёта регистра | все файлы |
| `--block-size`     | `-b`          | Размер блока для хеширования (байт)                   | 4096         |
| `--hash`           | `-a`          | Алгоритм хеширования: `crc32` или `md5`              | crc32        |
| `--help`           | `-h`          | Показать справку                                      | —            |

## Примеры

```bash
# Найти дубли в двух папках рекурсивно
bayan -d /home/user/docs -d /home/user/backup

# Искать только .txt файлы, минимальный размер 100 байт, MD5
bayan -d ./data -n "*.txt" -m 100 -a md5

# Только верхний уровень папки, блок 512 байт
bayan -d ./photos -l 0 -b 512

# Исключить папку tmp
bayan -d /home/user -e /home/user/tmp
```

## Формат вывода

Дубли печатаются группами: пути к файлам каждой группы через перевод строки,
между группами — пустая строка.

```
/path/to/file_a.txt
/path/to/copy_of_a.txt

/path/to/image.png
/backup/image.png
/archive/image.png
```
